# 🎛️ Botões Neomórficos

Projeto de botões estilizados com efeito neomorfismo, trazendo um visual moderno baseado em luz e sombra suave.

---

## 🚀 Tecnologias utilizadas

- HTML5  
- CSS3  

---

## 🎯 Objetivo

Explorar técnicas de design utilizando CSS para criar efeitos visuais modernos, incluindo:

- Uso de sombras externas e internas
- Criação de profundidade em elementos
- Estilização de botões interativos
- Aplicação do conceito de neomorfismo

---

## 🧩 Funcionalidades

- Botões com efeito visual 3D  
- Animação ao clicar (efeito pressionado)  
- Layout centralizado  
- Design limpo e minimalista  

---

## ✨ Diferenciais técnicos

- Uso de `box-shadow` para criar efeito de relevo  
- Simulação de luz e profundidade (neomorfismo)  
- Transição de estado com `:active`  
- Interface leve sem uso de JavaScript
